Import

Base mondiale de bornes de recharge — plus complète qu'OSM. Clé API gratuite sur openchargemap.io (compte requis).

Importe n'importe quel jeu de données JSON/GeoJSON depuis data.gouv.fr ou une autre source open data.

Jeux de données suggérés (clic pour remplir l'URL)

⚡ Bornes IRVE — Recharge électrique France ⚠ 545 MB 🚻 Toilettes publiques — France entière (~61 MB GeoJSON) 💧 Fontaines à boire — Paris 🚿 Sanisettes — Paris (OpenData Paris) 🛍️ Marchés découverts — Paris 🏖️ Plages — Métropole Nice (GeoJSON)

⚠ Les fichiers > 50 MB peuvent être trop lourds pour le navigateur. Préférer OSM ou OpenChargeMap pour les gros volumes.

Mapping des colonnes

Standard universel des opérateurs de mobilité partagée (vélos, trottinettes, scooters). Fonctionne avec Vélib', Lime, Bird, Tier, Dott, Bolt, Pony et tout opérateur publiant du GBFS.

Feeds préconfigurés (clic pour remplir)

🚲 Vélib' — Paris (stations fixes) 🚲 Vélo'v — Lyon (stations fixes) 🚲 VCub — Bordeaux (stations fixes) 🚲 Bicloo — Nantes (stations fixes) 🚲 Star Vélo — Rennes (stations fixes)

Pour Lime, Bird, Tier → cherche le feed de ta ville sur transport.data.gouv.fr ou gbfs.mobilitydata.org

⚠ Les opérateurs free-floating (pas de stations fixes) donnent des positions temps réel — non importables. Seuls les systèmes à stations fixes sont utiles ici.
